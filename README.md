## 📌 규칙

- 매주 화요일 2시 - Study Day
- 매주 목표 토픽 5개
- 토픽 관련 공식 문서가 있다면 꼭 읽기
- 각 토픽 당 서로에게 내줄 핵심 문제 1~2개 이슈에 등록하여 관리
    - 서로의 문제에 대한 답변은 해당 이슈에 댓글로 달기
- 토픽 관련 정리 내용은 본 Repository에 `주제/githubID.md`로 등록

```markdown
// githubID.md에 들어가야 하는 내용
토픽 제목
공부 내용
해당 토픽 내용이 면접 질문으로 나왔을 때 1분 가량의 요약 답변
```
